## 📋 启动步骤

### 终端1：启动后端

**新建终端窗口**，复制粘贴执行：

```bash
cd /Users/Python项目/HR人事/backend
export PATH="$HOME/Library/Python/3.12/bin:$PATH"
poetry run uvicorn app.main:app --reload --host 0.0.0.0 --port 9000
```

✅ 看到 `Uvicorn running on http://0.0.0.0:9000` 就说明成功了

---

### 终端2：启动前端

**再新建一个终端窗口**，复制粘贴执行：

```bash
cd /Users/Python项目/HR人事/frontend
npm run dev
```

✅ 看到 `Local: http://localhost:5173/` 就说明成功了

---

### 终端3：启动 ngrok（获取公网地址）

**再新建第三个终端窗口**，复制粘贴执行：

```bash
ngrok http 5173
```

✅ 会显示类似这样的信息：

```

Forwarding                    https://abc-123-xyz.ngrok-free.app -> http://localhost:5173
```

## 🔍 故障排查

### 问题1：后端启动失败
**提示**：端口 9000 被占用

**解决**：
```bash
# 查找占用进程
lsof -ti:9000

# 停止进程（替换为实际的进程ID）
kill -9 进程ID
```

### 问题2：前端启动失败
**提示**：端口 5173 被占用

**解决**：
```bash
# 查找占用进程
lsof -ti:5173

# 停止进程
kill -9 进程ID
```

### 问题3：HR 无法访问
**可能原因**：
1. ngrok 未正确启动
2. 地址复制错误
3. HR 的网络问题

**解决**：
- 检查 ngrok 终端是否显示 `Forwarding` 地址
- 重新复制地址发给 HR
- 让 HR 尝试用手机热点访问
