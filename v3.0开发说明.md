## 🎯 v3.0 核心特性

### 1. 现代化 PC 端布局
- 左侧深色导航栏（固定宽度 240px）
- 顶部白色导航栏（固定高度 60px）
- 主内容区域（自适应）
- 最小宽度：1280px（PC专用）

### 2. 6大功能模块
1. 📊 数据概览 (dashboard)
2. 👥 候选人管理 (candidates)
3. 📁 简历库 (resumes)
4. 📈 智能分析 (analytics)
5. ⚙️ 系统设置 (settings)
6. 📄 测评管理 (assessments)

### 3. 简历导入功能
- 支持 PDF、图片、Word、Excel
- 前端解析（PDF.js + Tesseract.js）
- 信息自动提取
- 简历与测评关联

### 4. 综合分析
- 简历 + EPQ 测评双维度分析
- 技术匹配度 + 性格匹配度
- 全景候选人画像
- 智能面试建议

## 🔧 技术栈

### 核心技术
- HTML5 + CSS3
- 原生 JavaScript (ES6+)
- LocalStorage (数据存储)

### 第三方库（CDN）
```html
<!-- 图表库 -->
<script src="https://cdn.jsdelivr.net/npm/echarts@5.4.3/dist/echarts.min.js"></script>

<!-- 图标库 -->
<link href="https://cdn.jsdelivr.net/npm/remixicon@3.5.0/fonts/remixicon.css" rel="stylesheet">

<!-- PDF 解析 -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js"></script>

<!-- OCR 识别（可选） -->
<script src="https://cdn.jsdelivr.net/npm/tesseract.js@4.1.1/dist/tesseract.min.js"></script>
```
