# EPQ 人才测评系统 v3.0 使用指南

## 🚀 快速开始

### 1. 部署到免费服务器

#### 方案A：Vercel（推荐）⭐⭐⭐⭐⭐

**优势：**
- 完全免费
- 自动 HTTPS
- 全球 CDN 加速
- 一键部署

**步骤：**
```bash
1. 注册 Vercel 账号
   访问：https://vercel.com/signup
   
2. 安装 Vercel CLI
   npm install -g vercel
   
3. 在项目目录执行
   cd /Users/hf/Desktop/临时/test
   vercel
   
4. 按提示操作，几分钟后获得访问链接
   例如：https://epq-test.vercel.app
```

#### 方案B：Netlify ⭐⭐⭐⭐

**步骤：**
```bash
1. 访问 https://app.netlify.com/
2. 点击 "Add new site" → "Deploy manually"
3. 拖拽整个文件夹上传
4. 获得访问链接
   例如：https://epq-test.netlify.app
```

#### 方案C：GitHub Pages ⭐⭐⭐

**步骤：**
```bash
1. 创建 GitHub 仓库
   仓库名：epq-system
   
2. 上传所有文件
   git init
   git add .
   git commit -m "v3.0 release"
   git push
   
3. 开启 GitHub Pages
   Settings → Pages → Source: main branch
   
4. 访问链接
   https://你的用户名.github.io/epq-system
```

### 2. 候选人管理

**用途：** 管理所有候选人

**功能：**
- **列表查看** - 所有候选人的基本信息
- **状态筛选** - 待测评/已测评/已面试/已录用
- **搜索** - 按姓名、手机号、岗位搜索
- **详情查看** - 点击候选人查看完整信息

**操作流程：**
```
1. 点击左侧 "👥 候选人管理"
2. 使用顶部筛选器筛选
3. 点击任意候选人查看详情
4. 在详情页可以：
   - 查看 EPQ 测评结果
   - 查看简历信息
   - 查看综合匹配度
   - 添加备注
   - 修改状态
```

---

### 3. 简历库 🆕

**用途：** 上传和管理候选人简历

**支持格式：**
- ✅ PDF 文档
- ✅ 图片（JPG, PNG）
- ✅ Word 文档（.docx）
- ✅ Excel 表格

**操作步骤：**

#### 3.1 上传简历
```
1. 点击左侧 "📁 简历库"
2. 点击 "上传简历" 按钮
3. 选择文件或拖拽上传
4. 等待解析完成
5. 确认或修改解析的信息
6. 关联到候选人（如果已测评）
```

#### 3.2 查看简历
```
1. 在简历库列表中点击任意简历
2. 查看：
   - 原始文件预览
   - 提取的信息（教育、经验、技能）
   - 技术匹配度
   - 是否已关联测评
```

#### 3.3 关联测评
```
方式1：自动关联
- 上传简历时，根据姓名+手机号自动匹配

方式2：手动关联
- 在简历详情页，选择对应的测评记录
- 点击 "关联" 按钮
```

---

### 4. 智能分析 🆕🔥

**用途：** 基于简历+测评的综合分析

**分析维度：**

#### 4.1 岗位匹配度分析
```
综合匹配度 = 技术匹配(30%) + 经验匹配(25%) + 性格匹配(25%) + 稳定性(10%) + 潜力(10%)

技术匹配度：
- 简历中的技能 vs 岗位要求
- 项目经验的相关度

经验匹配度：
- 工作年限是否符合
- 行业经验是否相关

性格匹配度：
- EPQ 测评结果
- 与岗位理想模型的契合度

稳定性：
- 工作频率（跳槽次数）
- EPQ N 分数

潜力：
- 学习能力
- 成长轨迹
```

#### 4.2 全景候选人画像
```
包含：
- 基本信息卡片
- 技术能力雷达图
- EPQ 性格雷达图
- 优势和风险点
- 面试建议
- 团队契合度
```

#### 4.3 竞争力对比
```
功能：
- 同岗位候选人排名
- S/A/B/C/D 五级评分
- Top 10 优质候选人推荐
```

**操作步骤：**
```
1. 点击左侧 "📈 智能分析"
2. 选择分析类型：
   - 岗位匹配度分析
   - 竞争力排名
   - 候选人画像
3. 选择要分析的候选人
4. 查看详细分析报告
5. 导出 PDF 报告（可选）
```

## 🔐 数据安全

### LocalStorage 存储

所有数据存储在浏览器的 LocalStorage 中：

**优势：**
- ✅ 完全本地化，数据不上传服务器
- ✅ 隐私安全
- ✅ 无需数据库

**注意事项：**
- ⚠️ 清空浏览器数据会丢失所有记录
- ⚠️ 定期导出备份
- ⚠️ 同一浏览器共享数据（不同标签页同步）

### 数据备份

**方法1：导出 CSV**
```
1. 在候选人管理页面
2. 点击 "导出 CSV" 按钮
3. 保存到本地
```

**方法2：导出 JSON**
```
1. 打开 debug.html
2. 点击 "查看数据"
3. 复制 JSON 内容
4. 保存为 .json 文件
```

**方法3：浏览器开发者工具**
```
1. 按 F12 打开开发者工具
2. 切换到 Application 标签
3. 左侧 Local Storage
4. 右键复制内容
```

## 🐛 常见问题

### Q1: 部署后页面空白？
**A:** 检查浏览器控制台（F12）是否有错误，通常是 CDN 资源加载失败，尝试刷新或更换网络。

### Q2: 上传简历后没有反应？
**A:** 
1. 检查文件大小（建议 <5MB）
2. 检查文件格式是否支持
3. 打开浏览器控制台查看错误

### Q3: 数据没有保存？
**A:** 
1. 检查浏览器是否启用 LocalStorage
2. 检查是否在隐私模式
3. 清空缓存后重试

### Q4: 简历解析不准确？
**A:** 
1. PDF 解析准确率约 70-80%
2. 建议手动确认和修改
3. 未来版本会引入 AI 解析提升准确率

### Q5: 如何分享给团队使用？
**A:** 
1. 部署后直接分享链接
2. 每个人使用相同的浏览器和设备
3. 或等待 v3.1 的多人协同功能
