## 🔧 技术说明

### 为什么数据可能不显示？

可能的原因：
1. **日期格式问题** - 旧版本使用 `toLocaleString()` 格式不标准
2. **数据结构缺失** - 某些字段为空
3. **浏览器兼容性** - 使用了不支持的 API
4. **JavaScript 错误** - 代码执行失败

### diagnostic.html 如何解决？

1. **自动检测** - 扫描所有潜在问题
2. **一键修复** - 自动修复日期和数据结构
3. **可视化反馈** - 清晰展示问题和结果
