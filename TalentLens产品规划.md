# TalentLens - 智能人才洞察平台

## 🎯 产品定位

**TalentLens** 是一款面向 HR 的智能人才筛选辅助系统，通过简历解析 + 多维测评 + AI 分析，为招聘决策提供数据支撑。

---

## 📊 核心价值

| 传统招聘痛点 | TalentLens 解决方案 |
|-------------|-------------------|
| 简历筛选耗时 | AI 自动解析提取 |
| 主观判断偏差 | 多维测评数据化 |
| 候选人对比困难 | 智能匹配排名 |
| 决策缺乏依据 | 全景画像分析 |

## 📋 功能模块

### 1. 简历中心

#### 1.1 简历上传
- **支持格式**：PDF、JPG/PNG、Word
- **上传方式**：拖拽上传、批量上传
- **存储管理**：简历库分类、标签管理

#### 1.2 AI 信息提取
- **基本信息**：姓名、联系方式、年龄
- **教育背景**：学历、学校、专业
- **工作经历**：公司、职位、时长
- **技能标签**：自动提取技术栈
- **项目经验**：关键项目摘要

#### 1.3 简历库管理
- **分类筛选**：按岗位、学历、经验
- **标签系统**：自定义标签
- **批量操作**：导出、删除、归档

---

### 2. 测评中心

#### 2.1 测评问卷库

| 测评类型 | 维度 | 题目数 | 适用场景 | 状态 |
|----------|------|--------|----------|------|
| **EPQ 人格测评** | E/N/P/L | 48 | 性格特质评估 | ✅ 已上线 |
| **MBTI 职业性格** | 16型人格 | 93 | 团队匹配分析 | 🚧 开发中 |
| **DISC 行为风格** | D/I/S/C | 28 | 沟通风格评估 | 🚧 开发中 |
| **职业价值观** | 8维度 | 40 | 文化匹配度 | 🚧 开发中 |
| **认知能力** | 逻辑/数字/语言 | 30 | 岗位能力评估 | 📋 规划中 |
| **自定义问卷** | HR 自定义 | 自定义 | 特定需求 | 📋 规划中 |

#### 2.2 测评管理
- **链接生成**：一键生成测评链接
- **进度追踪**：实时查看完成状态
- **结果统计**：自动计算分数

### 3. AI 分析引擎

#### 3.1 人才画像
```
┌─────────────────────────────────────┐
│           候选人全景画像             │

└─────────────────────────────────────┘
```

#### 3.2 岗位匹配算法
```javascript
匹配度 = 
  学历匹配 × 20% +
  经验匹配 × 25% +
  技能匹配 × 25% +
  性格匹配 × 30%
```

#### 3.3 AI 分析功能
- **性格分析**：基于测评结果的深度解读
- **优劣势分析**：SWOT 风格分析
- **面试建议**：针对性问题生成
- **发展预测**：职业发展方向建议

### 4. 决策支持

#### 4.1 匹配度评分
- **综合评分**：0-100 分
- **分维度评分**：学历/经验/技能/性格
- **评分说明**：AI 生成的评分解释

#### 4.2 候选人排名
- **智能排序**：按匹配度自动排名
- **对比分析**：多候选人横向对比
- **筛选推荐**：TOP N 推荐

#### 4.3 报告生成
- **个人报告**：单候选人详细报告
- **对比报告**：多候选人对比报告
- **导出格式**：PDF、Excel

## 💡 用户认证方案

根据您的需求，采用**简化认证**方案：

### 方案设计
```
┌─────────────────────────────────────┐
│           简化认证流程               │
├─────────────────────────────────────┤
│                                      │
│  1. 管理员设置系统密码               │
│     └─ 存储在环境变量/配置文件       │
│                                      │
│  2. 用户输入密码登录                 │
│     └─ 验证通过后创建 Session        │
│                                      │
│  3. Session 有效期                   │
│     └─ 24小时自动过期               │
│                                      │
│  4. 无需注册/找回密码等复杂流程      │
│                                      │
└─────────────────────────────────────┘
```

### 优点
- ✅ 实现简单
- ✅ 无需数据库用户表
- ✅ 适合内部使用
- ✅ 安全性足够（HTTPS + 密码）

app.secret_key = 'your-secret-key'

    if request.json.get('password') == ADMIN_PASSWORD:

def get_data():
    if not session.get('logged_in'):

## 🌍 国际参考

### 主流人才测评产品

| 产品 | 国家 | 核心功能 | 特点 |
|------|------|----------|------|
| **Workday** | 美国 | ATS + 测评 | 企业级全流程 |
| **HireVue** | 美国 | AI 视频面试 | 表情/语言分析 |
| **Pymetrics** | 美国 | 游戏化测评 | 神经科学基础 |
| **Criteria** | 美国 | 多维测评库 | 认知+性格+技能 |
| **PI** | 美国 | 行为测评 | 简洁高效 |
| **北森** | 中国 | 测评 + 管理 | 本土化 |
| **Moka** | 中国 | ATS + 测评 | 智能招聘 |

### 最佳实践
1. **多维度测评**：性格 + 能力 + 价值观
2. **数据融合**：简历 + 测评 = 完整画像
3. **AI 辅助**：智能分析，而非替代决策
4. **可视化呈现**：图表 + 报告，易于理解
5. **隐私保护**：数据安全，合规使用
