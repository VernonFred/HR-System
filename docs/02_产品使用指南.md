## 📖 系统定位

### 你会用到的核心功能

```
日常场景: 每天面试3-5人

传统方式:
├─ 阅读简历 (30分钟/人)
├─ 凭感觉判断性格
├─ 面试提问无重点 (1小时/人)
└─ 决策靠经验和直觉

使用本系统:
├─ 候选人完成EPQ等专业测评 (5分钟)
├─ AI生成人才画像 (10秒)
├─ 查看测评画像+AI分析 (5分钟)
├─ 面试有针对性提问 (40分钟)
└─ 决策有数据支撑

节省时间: 40分钟/人 × 5人 = 3.3小时/天
```

### 核心价值

**不是替代你的判断，而是帮你更快做出更好的判断**

- 🎯 **快速了解性格**: 测评画像30秒看懂候选人
- 🤖 **AI专家建议**: 2分钟阅读AI深度分析
- 💡 **面试有重点**: AI提示你该问什么问题
- 📊 **决策有依据**: 不只凭感觉，有数据支撑

## 🚀 5分钟快速上手

### 第一步：让候选人完成EPQ测评 (推荐)

**为什么选EPQ？**
- ⏱️ **最快**: 48题，5分钟完成
- 📊 **够用**: 4个维度看懂性格（外向性、稳定性、独立性、真实性）
- 🎯 **实用**: 能判断"适不适合这个岗位"
- ✅ **防作弊**: L维度检测候选人是否刻意美化

---

#### 操作步骤

**1. 生成测评链接**
```
进入"专业测评"页面
    ↓
点击"EPQ人格测评"卡片
    ↓
点击"分发"按钮
    ↓
配置参数:
  - 有效期: 7天 (推荐)
  - 表单字段: ✅ 姓名、✅ 手机号、✅ 应聘岗位
    ↓
生成专属链接
```

**2. 发送给候选人**
```
方式1: 微信发送链接 (最常用)
方式2: 短信发送链接
方式3: 邮件发送链接
```

**3. 候选人填写**
```
候选人点击链接
    ↓
填写基本信息 (姓名、手机、岗位)
    ↓
完成48道题目 (5分钟)
    ↓
提交
    ↓
系统自动生成画像
```

**💡 发送话术模板**:

#### 画像卡片重点关注的3个区域

**🎯 区域1: EPQ象限图** (30秒看懂性格)

```
┌─────────────────────────────────┐
│      EPQ 人格特质分布            │
│                                 │
│         外向性(E)               │
│            /\                   │
│           /  \                  │
│  掩饰性(L)    神经质(N)          │
│           \  /                  │
│            \/                   │
│         精神质(P)               │
│                                 │
│  ✅ 外向性: 85分 (外向、善沟通)   │
│  ✅ 神经质: 25分 (情绪稳定)      │
│  ⚠️ 掩饰性: 78分 (可能美化)      │
└─────────────────────────────────┘
```

**如何快速判断**:
- **外向性高 (>70)**: 适合客服、销售、市场
- **外向性低 (<40)**: 适合技术、研发、数据
- **神经质高 (>70)**: 情绪敏感，不适合高压岗位
- **神经质低 (<30)**: 情绪稳定，抗压能力强
- **掩饰性高 (>75)**: 可能刻意美化，面试需验证

---

**🤖 区域2: AI综合分析** (2分钟深度了解)

**AI会告诉你**:

1️⃣ **性格特质总结**
```
示例:
"候选人展现出明显的外向特质，善于与人沟通，在团队中能够
积极主动地表达观点。情绪稳定性较高，面对压力时能保持冷静，
适合需要频繁沟通和快速决策的工作环境。"
```

2️⃣ **适合什么岗位**
```
示例:
✅ 推荐岗位: 客服专员、销售代表、市场运营
❌ 不推荐: 后端研发、数据分析（需要长时间独立工作）
```

3️⃣ **优势亮点**
```
示例:
• 沟通能力强，能快速建立客户信任
• 情绪稳定，不易被客户投诉影响
• 适应能力强，能快速融入新团队
```

4️⃣ **潜在风险** ⭐ **重点看**
```
示例:
⚠️ 可能对重复性工作缺乏耐心
⚠️ 执行细节需要关注，建议面试验证
⚠️ 掩饰性较高，部分回答可能有美化倾向
```

5️⃣ **面试建议** ⭐ **最有价值**
```
示例:
📋 重点验证问题:
• "描述一次处理困难客户的经历"（验证情绪管理）
• "你如何应对重复性工作？"（验证耐心）
• "为什么选择客服岗位？"（验证职业动机）
```

### 第三步：带着AI建议去面试

#### AI帮你准备的3个问题

**根据AI分析的"潜在风险"准备面试问题**

**示例场景**: AI提示"可能对重复性工作缺乏耐心"

**面试问题**:
1. "你之前的工作中，有哪些是重复性的工作？你是如何应对的？"
2. "如果每天要处理大量相似的客户咨询，你会怎么保持工作热情？"
3. "描述一次你需要长期坚持做一件枯燥工作的经历。"

**观察点**:
- ✅ 回答具体，有实际案例 → AI分析可能不准，候选人能胜任
- ⚠️ 回答模糊，避重就轻 → AI分析准确，需要谨慎考虑

---

#### 面试中观察验证

**EPQ显示"外向性高"**
- 面试中观察: 是否主动沟通、眼神交流、表达流畅
- ✅ 一致 → EPQ可信
- ❌ 不一致 → 可能测评不认真，或现场紧张

**EPQ显示"掩饰性高"**
- 面试中观察: 回答是否过于完美、缺乏真实感
- 追问具体细节，看是否有破绽

```
✅ EPQ性格匹配
✅ AI分析推荐该岗位

```


```
❌ EPQ不匹配
❌ AI不推荐该岗位
❌ 面试表现一般
───────────────────
→ 不推荐
```

## 🎯 进阶功能：提升准确度

### 功能1: 上传简历

**为什么？**
- AI可以结合简历验证性格推断
- 岗位推荐更准确（基于工作经历）
- 综合评分更有参考价值

**操作**:
```
人员画像页面 → 候选人卡片 → "上传简历"按钮
    ↓
选择候选人的简历文件 (PDF/Word)
    ↓
AI自动解析简历
    ↓
点击"重新生成画像"
    ↓
AI生成更完整的画像
```

**效果对比**:

```
只有EPQ:
AI: "性格外向，推荐销售岗"
准确度: 75%

EPQ + 简历:
AI: "性格外向 + 有2年销售经验 + 业绩优秀，
     推荐ToB销售岗位，尤其适合企业服务行业"
准确度: 90%
```

---

### 功能2: 要求候选人做多个测评

**什么时候需要？**
- 重要岗位（如管理岗）
- 高薪岗位
- 有多个候选人需要对比

**推荐组合**:

| 岗位类型 | 推荐测评组合 | 时间 | 价值 |
|---------|------------|------|------|
| 客服/销售 | EPQ + DISC | 13分钟 | 看性格+沟通风格 |
| 技术/产品 | EPQ + MBTI | 20分钟 | 看稳定性+思维方式 |
| 管理岗位 | MBTI + DISC + EPQ | 28分钟 | 全面评估 |

**操作**:
```
初筛: 只要求EPQ (5分钟)
    ↓
通过初筛后
    ↓
复试前: 补充DISC或MBTI
    ↓
AI生成更完整的画像
```

**效果**:
- AI会进行"交叉验证"（两个测评互相验证）
- 在"评分详情"中显示"一致性得分"
- 准确度从80% → 85%

---

### 功能3: 配置岗位画像

**什么时候需要？**
- 经常招聘相同岗位（如客服、销售）
- 想让"岗位匹配分"更准确

**快速操作**:
```
左侧导航 → "岗位画像配置"
    ↓
点击"创建岗位画像"
    ↓
填写:
  - 岗位名称: 客服专员
  - 所属部门: 客服部
    ↓
点击"AI一键配置"
    ↓
AI自动生成能力维度和权重:
  ✅ 沟通能力 (25%)
  ✅ 情绪管理 (20%)
  ✅ 服务意识 (20%)
  ✅ 抗压能力 (15%)
  ✅ 学习能力 (10%)
  ✅ 团队协作 (10%)
    ↓
确认保存 (也可以微调)
```

**配置后的效果**:
- 候选人的"岗位匹配分"更准确
- AI推荐岗位时优先推荐你配置的岗位
- 不再推荐不相关的岗位

## ⚠️ 常见问题速查

### Q1: 候选人说链接打不开？

**排查步骤**:
1. 检查链接是否过期 (在"专业测评"页面查看)
2. 让候选人换浏览器试试 (推荐用微信打开)
3. 重新生成链接发送

---

### Q2: AI分析感觉不准？

**可能原因**:
1. **候选人测评不认真**
   - 查看EPQ的"掩饰性(L)"维度
   - 如果L>75，可能刻意美化
   - 解决: 面试重点验证

2. **数据太少** (只有EPQ)
   - 单个测评准确度约75%
   - 解决: 补充DISC或MBTI，上传简历

3. **AI模型选择**
   - 切换到"专家分析"试试
   - 或点击"重新生成"

**记住**: AI是辅助，不是答案，最终判断靠面试

---

### Q3: 综合评分很低怎么办？

**别慌！只有EPQ时，评分偏低很正常**

原因:
```
综合评分 = 测评分×40% + 岗位匹配×30% + 完整度×15% + 简历质量×15%

只有EPQ时:
- 测评分: 85分 (只有EPQ)
- 岗位匹配: 70分 (基于EPQ推断)
- 完整度: 65分 (只1项测评)
- 简历质量: 0分 (无简历)

→ 综合评分: 68分
```

**解决方案**:
- **不要只看综合评分**
- **重点看EPQ象限图和AI分析**
- 如需提升评分: 补充测评+上传简历

---

### Q4: AI功能突然不能用了？

**检查Token状态**:
```
进入"系统设置"
    ↓
查看"AI模型配置"
    ↓
如果显示"Token过期"
    ↓
去ModelScope获取新Token
    ↓
粘贴保存
```

**临时方案**:
- Token失效后，系统自动降级到"规则分析"
- 画像卡片会显示"规则分析"标识
- 功能仍可使用，但分析深度降低

---

### Q5: 画像导出的PDF模糊？

**解决方案**:
1. 确保浏览器缩放为100% (按Ctrl/Cmd + 0)
2. 等待画像完全加载 (象限图显示完整)
3. 重新导出

**推荐**:
- 日常分享用PNG (清晰快速)
- 正式汇报用PDF (格式规范)

---

**注意**:

### ✅ 推荐做法

   - 综合评分更有参考价值

   - 一次配置，长期使用
   - 岗位匹配更准确
   - 用"AI一键配置"很快

5. **面试前必看AI的"面试建议"**
   - 这是最有价值的部分
   - 帮你准备针对性问题
   - 提高面试效率

### ❌ 不推荐做法

1. ❌ 只看综合评分就决定录不录用
   - 评分只是参考
   - 重点看EPQ和AI分析

2. ❌ 完全依赖AI的结论
   - AI是辅助，不是答案
   - 最终判断靠面试

3. ❌ 不看AI分析，只看雷达图
   - 雷达图只是数据
   - AI分析才是洞察

   - 建议7-14天
   - 避免候选人来不及填

   - AI的"潜在风险"很有价值
   - 面试必须验证
