## 🚀 快速开始

### 1. 环境准备

```bash
# 检查环境
python3 --version  # 需要 3.10+
node --version     # 需要 18+
sqlite3 --version  # 开发环境（生产推荐PostgreSQL）
```

### 2. 安装依赖

```bash
# 后端（推荐使用 Poetry）
cd backend
poetry install  # 或使用 pip install -r requirements.txt

# 前端
cd frontend
npm install
```

### 3. 启动服务

```bash
# 后端 (终端1)
cd backend
poetry run uvicorn app.main:app --reload --port 9000
# 或
python3 -m venv venv && source venv/bin/activate
uvicorn app.main:app --reload --port 9000

# 前端 (终端2)
cd frontend
npm run dev
```

### 4. 访问系统

- 前端: http://localhost:5173
- 后端 API: http://localhost:9000
- API 文档: http://localhost:9000/docs
- 默认账号: `admin` / `admin123`

## 📋 系统功能

| 模块 | 功能 | 新特性 |
|------|------|--------|
| 人员画像 | 候选人综合画像展示 | ✨ 多测评交叉验证、AI降级算法、简历质量评分 |
| 岗位推荐 | AI深度岗位匹配分析 | ✨ 动态岗位画像、场景化推荐（组织阶段/团队/上级风格） |
| 岗位画像配置 | AI 辅助岗位能力配置 | 支持JD一键解析、动态权重配置 |
| 专业测评 | MBTI/DISC/EPQ 测评管理 | 支持候选人端答题、实时计分 |
| 问卷中心 | 自定义问卷创建和管理 | 支持评分/调查两种模式 |
| 人员管理 | 人员数据统计和管理 | 简历上传、测评记录查看 |
| 系统设置 | 账户安全、AI 配置 | ModelScope API配置 |

### ✨ 新增功能亮点 (v2.0)

#### P1-1: 多测评交叉验证
- **一致性分析**: 跨MBTI/DISC/EPQ的特质一致性检验
- **置信度评分**: 高/中/低三级置信度标识
- **矛盾检测**: 自动识别测评结果中的不一致点

#### P1-2: AI降级算法
- **规则引擎**: AI超时/失败时自动切换基于规则的分析
- **状态指示**: UI明确显示AI分析/规则分析
- **重试机制**: 允许HR手动重试AI分析

#### P2-2: 简历质量评分
- **多维度评估**: 完整性(40%) + 逻辑性(30%) + 相关性(30%)
- **优势识别**: 自动提取简历亮点
- **改进建议**: 给出简历优化方向

#### P2-3: 岗位推荐算法优化
- **动态岗位画像**: 从数据库加载，支持HR自定义配置
- **场景化推荐**: AI分析"什么阶段的组织、什么类型的团队、什么风格的上级"
- **深度洞察**: 不只是岗位列表，而是适配场景分析

## 🔐 安全注意事项

1. **生产环境必须修改默认密码**：`admin/admin123`
2. **配置环境变量**：在 `.env` 中配置 `MODELSCOPE_API_KEY`

4. **HTTPS部署**：生产环境使用Nginx + SSL证书
